# 📊 Quant Portfolio Simulator

A Streamlit-based portfolio tool that lets users build and compare investment strategies using real market data.

---

## 🚀 What it does

This app allows users to:
- Build a portfolio from a stock universe
- Choose between different strategies:
  - Manual allocation
  - Momentum strategy
  - Risk parity strategy
- Compare performance against SPY
- View risk and return metrics
- Generate a PDF investment summary

---

## 📊 Features

- Portfolio construction
- Strategy comparison engine
- Equity curve visualization
- Sharpe ratio, volatility, drawdown metrics
- SPY benchmark comparison
- PDF export of results

---

## 🛠️ Tech Stack

- Python
- Streamlit
- yFinance
- Pandas
- NumPy
- ReportLab

---
